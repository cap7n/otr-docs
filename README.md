# OTR Design Wiki

Design wiki for **OTR**, a 2-player co-op car game built in Godot 4.

**Live site:** https://cap7n.github.io/otr-docs/

## Editing

All content lives in `docs/` as Markdown. Push to `main` and GitHub Actions rebuilds the site automatically. See the [How to Edit](docs/project/how-to-edit.md) page for details.

## Local preview (optional)

```
pip install mkdocs-material
mkdocs serve
```
